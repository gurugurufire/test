sqrt

プロトタイプ

double sqrt(double x);

詳細説明

sqrt関数は引数として与えられた数値の平方根を計算して返します。引数は非負の値でなければなりません。

使用例

コード

#include <stdio.h>
#include <math.h>

int main() {
    double num = 9.0;
    double result = sqrt(num);
    printf("数値 %f の平方根は %f です。\n", num, result);
    return 0;
}

出力

数値 9.000000 の平方根は 3.000000 です。
