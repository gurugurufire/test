fscanf

プロトタイプ

int fscanf(FILE *stream, const char *format, ...);

詳細説明

fscanf関数は、指定されたファイルストリームからフォーマットされたデータを読み取り、引数に従って変数に格納します。成功すると、読み取った項目の数を返します。失敗するとEOFを返します。

使用例

コード

#include <stdio.h>

int main() {
    FILE *fp = fopen("data.txt", "r");
    if(fp == NULL) {
        perror("ファイルを開けませんでした");
        return -1;
    }
    
    int num;
    fscanf(fp, "%d", &num);
    printf("ファイルから読み取った数値: %d\n", num);
    
    fclose(fp);
    return 0;
}

出力

ファイルから読み取った数値: [ファイルに格納されている数値]
