rand

プロトタイプ

int rand(void);

詳細説明

rand関数は、疑似乱数を生成して0からRAND_MAXまでの範囲の整数を返します。より予測不可能な乱数列を生成するために、srand関数で乱数生成器をシードすることが推奨されます。

使用例

コード

#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    srand(time(NULL));  // 乱数生成器のシード
    int random_number = rand();  // 乱数の生成
    printf("生成された乱数: %d", random_number);
    return 0;
}

出力

生成された乱数: [ランダムな数値]
