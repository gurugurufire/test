isdigit

プロトタイプ

int isdigit(int c);

詳細説明

isdigit関数は、引数として与えられた文字が数字（0-9）であるかどうかをテストします。もし数字であれば、0以外の値（真）を返し、そうでなければ0（偽）を返します。

使用例

コード

#include <stdio.h>
#include <ctype.h>

int main() {
    char c;
    printf("文字を入力してください: ");
    scanf("%c", &c);

    if(isdigit(c)) {
        printf("'%c' は数字です。\n", c);
    } else {
        printf("'%c' は数字ではありません。\n", c);
    }

    return 0;
}
