wait

プロトタイプ

pid_t wait(int *status);

詳細説明

wait関数は、呼び出し元のプロセスが子プロセスの終了を待ちます。子プロセスが終了すると、そのプロセスIDが返され、statusポインタが終了情報で更新されます。

使用例

コード

#include <sys/wait.h>
#include <stdio.h>
#include <unistd.h>

int main() {
    pid_t pid = fork();
    if (pid == 0) {
        // 子プロセス
        printf("子プロセス: PID = %d\n", getpid());
    } else {
        // 親プロセス
        int status;
        pid_t childPid = wait(&status);
        printf("親プロセス: 子プロセス %d が終了しました\n", childPid);
    }
    return 0;
}

出力

子プロセス: PID = [子プロセスのPID]
親プロセス: 子プロセス [子プロセスのPID] が終了しました
