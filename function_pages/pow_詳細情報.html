pow

プロトタイプ

double pow(double base, double exponent);

詳細説明

pow関数は、指定された基数（base）の指定された指数（exponent）に対する累乗計算を行います。

使用例

コード

#include <stdio.h>
#include <math.h>

int main() {
    double base = 2.0;
    double exponent = 3.0;
    double result = pow(base, exponent);
    
    printf("Result: %.2f", result);
    return 0;
}

出力

Result: 8.00
