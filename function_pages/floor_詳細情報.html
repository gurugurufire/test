floor

プロトタイプ

double floor(double x);

詳細説明

floor関数は、与えられた実数値x以下の最大の整数を返します。つまり、xを超えない最大の整数値を計算します。

使用例

コード

#include <stdio.h>
#include <math.h>

int main() {
    double x = 3.7;
    double result = floor(x);
    printf("%.2f以下の最大の整数は: %.2f\n", x, result);
    return 0;
}

出力

3.70以下の最大の整数は: 3.00
