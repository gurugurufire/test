strcat

プロトタイプ

char *strcat(char *dest, const char *src);

詳細説明

strcat関数は、第二引数のsrcで指定された文字列を第一引数のdestで指定された文字列の末尾に結合します。戻り値は結合された文字列へのポインタです。

使用例

コード

#include <stdio.h>
#include <string.h>

int main() {
    char str1[20] = "Hello";
    char str2[] = " World";
    strcat(str1, str2);
    printf("%s\n", str1);
    return 0;
}

出力

Hello World
