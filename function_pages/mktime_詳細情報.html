mktime

プロトタイプ

time_t mktime(struct tm *timeptr);

詳細説明

mktime関数は、struct tmによって指定されたローカル時刻を、time_t型のカレンダー時刻に変換します。この関数はヘッダーファイルに定義されています。

使用例

コード

#include <stdio.h>
#include <time.h>

int main() {
    struct tm t;
    time_t t_of_day;

    t.tm_year = 2023 - 1900;  // 年は1900年から数える
    t.tm_mon = 3;             // 月は0から数える
    t.tm_mday = 31;
    t.tm_hour = 0;
    t.tm_min = 0;
    t.tm_sec = 0;
    t.tm_isdst = -1;          // 夏時間は未設定
    t_of_day = mktime(&t);

    printf("seconds since the Epoch: %ld\n", (long)t_of_day);
    return 0;
}

出力

seconds since the Epoch: [エポックからの秒数が表示されます]
