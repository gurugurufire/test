free

プロトタイプ

void free(void *ptr);

詳細説明

free関数は、mallocやcallocなどで割り当てられたメモリ領域を解放します。ptrがNULLポインタの場合、何も行われません。

使用例

コード

#include <stdlib.h>

int main() {
    int *ptr = malloc(sizeof(int) * 5);
    if(ptr == NULL) {
        perror("メモリ割り当てに失敗しました");
        return -1;
    }

    // ptrを使用して何かを行う

    free(ptr); // 割り当てられたメモリを解放する
    return 0;
}

出力

メモリ割り当てと解放の例です。実際の出力はありません。
