strncmp

プロトタイプ

int strncmp(const char *s1, const char *s2, size_t n);

詳細説明

strncmp関数は、s1とs2の最初のn文字を比較し、比較の結果に基づいて整数を返します。s1がs2より辞書的に小さい場合は負の整数、等しい場合は0、大きい場合は正の整数を返します。

使用例

コード

#include <stdio.h>
#include <string.h>

int main() {
    char s1[] = "Hello";
    char s2[] = "World";
    int result = strncmp(s1, s2, 3);
    printf("結果: %d", result);
    return 0;
}

出力

結果: -15
