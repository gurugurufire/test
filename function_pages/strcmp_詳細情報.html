strncmp

プロトタイプ

int strncmp(const char *str1, const char *str2, size_t n);

詳細説明

strncmp関数は、2つの文字列str1とstr2を比較し、最大n文字まで比較します。返り値は、str1がstr2よりも辞書的に前にくる場合は負の整数、等しい場合は0、str1がstr2よりも辞書的に後ろにくる場合は正の整数を返します。

使用例

コード 

char str1[] = "apple";
char str2[] = "banana";
int result = strncmp(str1, str2, 5);
printf("結果: %d", result);

出力 

結果: -1 (str1がstr2よりも辞書的に前にくる場合)
