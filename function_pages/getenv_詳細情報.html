getenv

プロトタイプ

char *getenv(const char *name);

詳細説明

getenv関数は、引数で指定された名前の環境変数の値を返します。環境変数が存在しない場合はNULLを返します。

使用例

コード

#include <stdio.h>
#include <stdlib.h>

int main() {
    const char *env_var = getenv("PATH");
    if(env_var != NULL) {
        printf("PATH: %s\n", env_var);
    } else {
        printf("PATH環境変数が見つかりませんでした。\n");
    }
    return 0;
}

出力

PATH: [システムのPATH環境変数の値]
