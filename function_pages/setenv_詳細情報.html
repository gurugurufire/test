setenv

プロトタイプ

int setenv(const char *name, const char *value, int overwrite);

詳細説明

setenv関数は、指定された名前の環境変数を新しい値に設定します。もし環境変数が既に存在していて、overwriteが0でなければ、値は上書きされます。成功時に0を、エラーが発生した場合には-1を返します。

使用例

コード

#include <stdlib.h>
#include <stdio.h>

int main() {
    if (setenv("MY_VAR", "123", 1) != 0) {
        perror("setenv failed");
    } else {
        printf("MY_VAR set to %s\n", getenv("MY_VAR"));
    }
    return 0;
}

出力

MY_VAR set to 123
