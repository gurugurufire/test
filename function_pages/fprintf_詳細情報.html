fprintf

プロトタイプ

int fprintf(FILE *stream, const char *format, ...);

詳細説明

fprintf関数は、指定されたストリームに対して書式付きの出力を行います。第一引数は出力先のストリーム、第二引数は書式文字列を指定します。 その後の引数は、書式文字列内の書式指定子に対応する変数を順に指定します。

使用例

コード

        FILE *fp = fopen("output.txt", "w");
        if (fp) {
            fprintf(fp, "Hello, %s!", "World");
            fclose(fp);
        }

出力

        // output.txtファイルの内容:
        Hello, World!
