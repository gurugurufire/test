strrchr

プロトタイプ

char *strrchr(const char *str, int c);

詳細説明

strrchr関数は、strの中で文字cが最後に出現する位置を探します。見つかった場合は、その位置へのポインタを返します。もし見つからない場合はNULLを返します。

使用例

コード

#include <stdio.h>
#include <string.h>

int main() {
    const char *str = "Hello World";
    char *last_occurrence = strrchr(str, 'o');
    if (last_occurrence != NULL) {
        printf("最後の'o'の位置: %ld\n", last_occurrence - str);
    } else {
        printf("'o'は見つかりませんでした。\n");
    }
    return 0;
}

出力

最後の'o'の位置: 7
