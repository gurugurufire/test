strcpy

プロトタイプ

char *strcpy(char *dest, const char *src);

詳細説明

strcpy関数は、srcで指定された文字列をdestにコピーします。コピーされた文字列の終端にはヌル文字('\0')が追加されます。

使用例

コード

#include <stdio.h>
#include <string.h>

int main() {
    char src[] = "Hello World";
    char dest[20];
    strcpy(dest, src);
    printf("コピーされた文字列: %s\n", dest);
    return 0;
}

出力

コピーされた文字列: Hello World
