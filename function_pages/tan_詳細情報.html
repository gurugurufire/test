tan

プロトタイプ

double tan(double x);

詳細説明

tan関数は、引数として与えられた角度（ラジアン単位）の正接（タンジェント）を計算して返します。

使用例

コード

#include <stdio.h>
#include <math.h>

int main() {
    double angle = M_PI / 4; // 45度（ラジアン単位）
    double result = tan(angle);
    printf("角度: %lf の正接は: %lf\n", angle, result);
    return 0;
}

出力

角度: 0.785398 の正接は: 1.000000
