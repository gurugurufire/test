gets

プロトタイプ

char *gets(char *str);

詳細説明

gets関数は、標準入力から改行文字までの文字列を読み取り、引数で指定された文字列に格納します。入力の終わりに達するか、エラーが発生するとNULLを返します。

警告

gets関数は安全でないため、使用しないことが推奨されます。代わりにfgets関数を使用してください。

使用例

コード

#include <stdio.h>

int main() {
    char str[100];
    printf("文字列を入力してください: ");
    gets(str);
    printf("入力された文字列: %s\n", str);
    return 0;
}

出力

[入力された文字列]
