setlocale

プロトタイプ

char *setlocale(int category, const char *locale);

詳細説明

setlocale関数は、プログラムのロケールを設定またはクエリします。categoryパラメータは設定するロケールのカテゴリを指定し、localeパラメータは使用するロケールの名前を指定します。成功時には使用されているロケールの名前を、エラーが発生した場合にはNULLを返します。

使用例

コード

#include <locale.h>
#include <stdio.h>

int main() {
    char *result = setlocale(LC_ALL, "en_US.UTF-8");
    if (result == NULL) {
        printf("Locale not changed\n");
    } else {
        printf("Locale set to %s\n", result);
    }
    return 0;
}

出力

Locale set to en_US.UTF-8
