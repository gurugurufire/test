strncat

プロトタイプ

char *strncat(char *dest, const char *src, size_t n);

詳細説明

strncat関数は、src文字列の最初のn文字をdest文字列の終端に追加します。結果としてできる文字列はnullで終わります。

使用例

コード

#include <stdio.h>
#include <string.h>

int main() {
    char dest[20] = "Hello, ";
    char src[] = "World!";
    strncat(dest, src, 3);
    printf("%s", dest);
    return 0;
}

出力

Hello, Wor
