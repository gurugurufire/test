unsetenv

プロトタイプ

int unsetenv(const char *name);

詳細説明

unsetenv関数は、指定された名前の環境変数を削除します。成功した場合は0を、エラーが発生した場合は-1を返します。

使用例

コード

#include <stdio.h>
#include <stdlib.h>

int main() {
    const char* varName = "MY_VAR";
    setenv(varName, "123", 1);
    printf("環境変数 %s = %s\n", varName, getenv(varName));
    unsetenv(varName);
    printf("環境変数 %s = %s\n", varName, getenv(varName));
    return 0;
}

出力

環境変数 MY_VAR = 123
環境変数 MY_VAR = (null)
