strlen

プロトタイプ

size_t strlen(const char *s);

詳細説明

strlen関数は、null終端文字列sの長さを返します（null終端文字を含まない）。

使用例

コード

#include <stdio.h>
#include <string.h>

int main() {
    const char *str = "Hello, World!";
    size_t length = strlen(str);
    printf("文字列の長さ: %zu", length);
    return 0;
}

出力

文字列の長さ: 13
