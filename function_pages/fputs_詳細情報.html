fputs

プロトタイプ

int fputs(const char *str, FILE *stream);

詳細説明

fputs関数は、指定された文字列を指定されたストリームに書き込みます。成功した場合は非負の値を、エラーが発生した場合はEOFを返します。

使用例

コード

#include <stdio.h>

int main() {
    FILE *fptr;
    char text[] = "Hello, World!";

    fptr = fopen("example.txt", "w");
    if(fptr == NULL) {
        perror("ファイルを開けません");
        return -1;
    }

    fputs(text, fptr);
    fclose(fptr);

    return 0;
}

出力

example.txtファイルに"Hello, World!"が書き込まれます。
