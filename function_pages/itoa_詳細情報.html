itoa

プロトタイプ

char* itoa(int value, char* str, int base);

詳細説明

itoa関数は、整数を文字列に変換します。第一引数は変換したい整数、第二引数は結果の文字列を格納するためのバッファ、第三引数は数値の基数です。

使用例

コード

#include <stdio.h>
#include <stdlib.h>

int main() {
    int number = 123;
    char buffer[10];

    itoa(number, buffer, 10);
    printf("整数 %d の文字列は %s です。\n", number, buffer);

    return 0;
}

出力

整数 123 の文字列は 123 です。
