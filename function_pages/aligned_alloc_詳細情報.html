aligned_alloc

プロトタイプ

void* aligned_alloc(size_t alignment, size_t size);

詳細説明

aligned_alloc関数は、指定されたアライメントで指定されたサイズのメモリブロックを割り当てる関数です。成功するとアライメントされたメモリブロックへのポインタを返し、失敗するとNULLを返します。

使用例

コード

void* ptr = aligned_alloc(16, 100);  // 16バイトアライメントで100バイトのメモリを確保
if (ptr) {
    free(ptr);  // メモリを解放
}

出力

メモリが正しく割り当てられれば、それを解放します。
