atexit

プロトタイプ

int atexit(void (*func)(void));

詳細説明

atexit関数は、プログラムが正常に終了するときに実行される関数を登録するために使用されます。登録された関数は逆順で実行されます。

使用例

コード

#include <stdio.h>
#include <stdlib.h>

void goodbye(void) {
    printf("Goodbye!\n");
}

int main() {
    atexit(goodbye);
    printf("Hello!\n");
    return 0;
}

出力

Hello!
Goodbye!
