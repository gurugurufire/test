tolower

プロトタイプ

int tolower(int c);

詳細説明

tolower関数は、引数として与えられた文字が大文字の場合にそれを小文字に変換します。変換された文字、または引数が大文字でない場合は引数をそのまま返します。

使用例

コード

#include <stdio.h>
#include <ctype.h>

int main() {
    char ch = 'A';
    int lowerCh = tolower(ch);
    printf("大文字: %c, 小文字: %c\n", ch, lowerCh);
    return 0;
}

出力

大文字: A, 小文字: a
