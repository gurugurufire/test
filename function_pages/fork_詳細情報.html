fork

プロトタイプ

pid_t fork(void);

詳細説明

fork関数は新しいプロセスを作成します。この関数が成功すると、2つのプロセスが存在するようになります。呼び出し元のプロセスは「親プロセス」と呼ばれ、新しいプロセスは「子プロセス」と呼ばれます。子プロセスは親プロセスのコピーであり、ほとんどの状態が同じですが、異なるプロセスIDを持っています。

使用例

コード

#include <stdio.h>
#include <sys/types.h>
#include <unistd.h>

int main() {
    pid_t pid = fork();

    if (pid == 0) {
        // 子プロセス
        printf("これは子プロセスです。\n");
    } else if (pid > 0) {
        // 親プロセス
        printf("これは親プロセスです。子プロセスのPIDは %d です。\n", pid);
    } else {
        // forkに失敗
        perror("forkに失敗しました");
    }

    return 0;
}

出力

これは親プロセスです。子プロセスのPIDは [子プロセスのPID] です。
これは子プロセスです。
