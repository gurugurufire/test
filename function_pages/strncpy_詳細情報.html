strncpy

プロトタイプ

char *strncpy(char *dest, const char *src, size_t n);

詳細説明

strncpy関数は、srcの最初のn文字をdestにコピーします。もしsrcがn文字に満たない場合、残りの部分はnull文字で埋められます。

使用例

コード

#include <stdio.h>
#include <string.h>

int main() {
    char src[] = "Hello World";
    char dest[6];
    strncpy(dest, src, 5);
    dest[5] = '\0'; // 終端のnull文字を追加
    printf("コピーされた文字列: %s", dest);
    return 0;
}

出力

コピーされた文字列: Hello
