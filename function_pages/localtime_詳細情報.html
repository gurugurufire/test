localtime

プロトタイプ

struct tm* localtime(const time_t* timer);

詳細説明

localtime関数は、time_t型のポインタを引数として受け取り、それをローカル時刻に変換したtm構造体へのポインタを返します。この構造体は、年、月、日、時、分、秒などの情報を含んでいます。

使用例

コード

#include <stdio.h>
#include <time.h>

int main() {
    time_t rawtime;
    struct tm* timeinfo;

    time(&rawtime);
    timeinfo = localtime(&rawtime);
    printf("現在のローカル時刻: %s", asctime(timeinfo));

    return 0;
}

出力

現在のローカル時刻: Wed Jan  2 02:37:41 2019
