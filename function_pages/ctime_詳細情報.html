ctime

プロトタイプ

char *ctime(const time_t *timer);

詳細説明

ctime関数は、time_t型の値を受け取り、それを読みやすい文字列の形式に変換します。この関数は<time.h>ヘッダーに定義されています。

使用例

コード

#include <stdio.h>
#include <time.h>

int main() {
    time_t rawtime;
    time(&rawtime);
    printf("現在の日時: %s", ctime(&rawtime));
    return 0;
}

出力

現在の日時: Mon Jan 1 12:00:00 1970
