islower

プロトタイプ

int islower(int c);

詳細説明

islower関数は、引数として与えられた文字が小文字であるかどうかをテストします。もし小文字であれば、0以外の値（真）を返し、そうでなければ0（偽）を返します。

使用例

コード

#include <stdio.h>
#include <ctype.h>

int main() {
    char c;
    printf("文字を入力してください: ");
    scanf("%c", &c);

    if(islower(c)) {
        printf("'%c' は小文字です。\n", c);
    } else {
        printf("'%c' は小文字ではありません。\n", c);
    }

    return 0;
}
