strchr

プロトタイプ

char *strchr(const char *str, int c);

詳細説明

strchr関数は、第一引数のstrで指定された文字列内で、第二引数のcで指定された文字を検索します。見つかった場合はその文字へのポインタを返し、見つからなかった場合はNULLを返します。

使用例

コード

#include <stdio.h>
#include <string.h>

int main() {
    const char *str = "Hello World";
    char c = 'W';
    char *result = strchr(str, c);
    if(result != NULL) {
        printf("文字 '%c' が見つかりました: %s\n", c, result);
    } else {
        printf("文字 '%c' は見つかりませんでした。\n", c);
    }
    return 0;
}

出力

文字 'W' が見つかりました: World
