cos

プロトタイプ

double cos(double x);

詳細説明

cos関数は、引数として与えられた角度（ラジアン単位）の余弦（コサイン）を計算します。この関数は<math.h>ヘッダーに定義されています。

使用例

コード

#include <stdio.h>
#include <math.h>

int main() {
    double angle = M_PI / 3; // ラジアンで60度
    printf("cos(%lf) = %lf\n", angle, cos(angle));
    return 0;
}

出力

cos(1.047198) = 0.500000
