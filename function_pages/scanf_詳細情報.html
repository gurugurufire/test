scanf

プロトタイプ

int scanf(const char* format, ...);

詳細説明

scanf関数は、標準入力からフォーマットされたデータを読み取り、指定された変数に格納します。この関数は正常に読み取った入力項目の数を返します。

使用例

コード

#include <stdio.h>

int main() {
    int age;
    float height;
    printf("年齢と身長を入力してください: ");
    scanf("%d %f", &age, &height);
    printf("年齢: %d, 身長: %.2f", age, height);
    return 0;
}

出力

年齢と身長を入力してください: （ユーザーの入力によって異なります）
