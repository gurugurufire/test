perror

プロトタイプ

void perror(const char *str);

詳細説明

perror関数は、errno変数に設定された最後のエラーコードに対応するエラーメッセージを標準エラー出力に出力します。オプションで、エラーメッセージの前に追加の文字列を出力するためにstrを指定することができます。

使用例

コード

#include <stdio.h>
#include <errno.h>
#include <string.h>

int main() {
    FILE *fptr;

    fptr = fopen("nonexistentfile.txt", "r");
    if (fptr == NULL) {
        perror("Error");
    }
    return 0;
}

出力

Error: No such file or directory
