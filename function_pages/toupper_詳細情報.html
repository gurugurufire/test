toupper

プロトタイプ

int toupper(int c);

詳細説明

toupper関数は、引数として与えられた文字が小文字の場合にそれを大文字に変換します。変換された文字、または引数が小文字でない場合は引数をそのまま返します。

使用例

コード

#include <stdio.h>
#include <ctype.h>

int main() {
    char ch = 'a';
    int upperCh = toupper(ch);
    printf("小文字: %c, 大文字: %c\n", ch, upperCh);
    return 0;
}

出力

小文字: a, 大文字: A
