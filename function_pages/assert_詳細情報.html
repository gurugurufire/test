assert

プロトタイプ

void assert(int expression);

詳細説明

assertマクロは、指定された式が真であることを確認します。式が偽である場合、assertはエラーメッセージを表示し、プログラムを異常終了させます。

使用例

コード

#include <assert.h>
#include <stdio.h>

int main() {
    int a = 5;
    assert(a == 5);  // This will pass
    assert(a == 10);  // This will fail and abort the program
    
    return 0;
}

出力

Assertion failed: (a == 10), function main, file main.c, line 7.
