asctime

プロトタイプ

char *asctime(const struct tm *timeptr);

詳細説明

asctime関数は、指定されたstruct tm構造体を人間が読める文字列に変換します。この文字列は、曜日、月、日、時、分、秒、および年を表します。

使用例

コード

#include <stdio.h>
#include <time.h>

int main() {
    time_t rawtime;
    struct tm * timeinfo;

    time(&rawtime);
    timeinfo = localtime(&rawtime);
    
    printf("Current date and time: %s", asctime(timeinfo));
    
    return 0;
}

出力

Current date and time: Wed Oct 31 12:34:56 2023
