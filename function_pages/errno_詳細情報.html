errno

概要

errnoは、標準ライブラリ関数によって発生したエラーを報告するためのマクロまたは変数です。通常、エラーが発生した場合にセットされ、0にリセットされます。

ヘッダーファイル

<errno.h>

使用例

コード

#include <stdio.h>
#include <errno.h>
#include <string.h>

int main() {
    FILE *fp;
    
    fp = fopen("存在しないファイル.txt", "r");
    
    if (fp == NULL) {
        printf("エラー: %s\n", strerror(errno));
    }
    
    return 0;
}

出力

エラー: No such file or directory
