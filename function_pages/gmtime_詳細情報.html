gmtime

プロトタイプ

struct tm *gmtime(const time_t *timer);

詳細説明

gmtime関数は、UNIXエポックからの秒数を格納したtime_t型の値を取り、それをUTCを基準としたtm構造体に変換します。この関数は変換されたtm構造体へのポインタを返します。

使用例

コード

#include <stdio.h>
#include <time.h>

int main() {
    time_t rawtime;
    struct tm *timeinfo;

    time(&rawtime); // 現在の時間を取得
    timeinfo = gmtime(&rawtime); // UTCに変換

    printf("現在のUTC時間: %s", asctime(timeinfo));

    return 0;
}

出力

[現在のUTC時間]
