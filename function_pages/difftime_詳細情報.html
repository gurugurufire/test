difftime

プロトタイプ

double difftime(time_t time2, time_t time1);

詳細説明

difftime関数は、2つのtime_t型の値（time2およびtime1）間の時間差（秒単位）を計算します。この関数は<time.h>ヘッダーに定義されています。

使用例

コード

#include <stdio.h>
#include <time.h>

int main() {
    time_t start, end;
    double diff;
    
    time(&start);
    // 何か時間のかかる処理
    time(&end);
    
    diff = difftime(end, start);
    
    printf("処理にかかった時間: %f秒\n", diff);
    return 0;
}

出力

処理にかかった時間: x.xxxxxx秒
