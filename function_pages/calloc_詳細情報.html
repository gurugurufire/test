calloc

プロトタイプ

void *calloc(size_t num, size_t size);

詳細説明

calloc関数は、指定された数の要素に対して動的にメモリを割り当て、割り当てられたメモリを0で初期化します。成功すると、割り当てられたメモリへのポインタを返し、失敗するとNULLを返します。

使用例

コード

#include <stdio.h>
#include <stdlib.h>

int main() {
    int* arr = (int*) calloc(5, sizeof(int));
    if (arr != NULL) {
        for (int i = 0; i < 5; i++) {
            printf("%d ", arr[i]);
        }
        free(arr);
    }
    return 0;
}

出力

0 0 0 0 0
