system

プロトタイプ

int system(const char *command);

詳細説明

system関数は、指定されたコマンドラインをホスト環境のシェルまたはコマンドラインインタープリタに渡して実行します。コマンドの実行が成功した場合はコマンドの終了ステータスを返し、エラーが発生した場合は-1を返します。

使用例

コード

#include <stdio.h>
#include <stdlib.h>

int main() {
    int return_value;
    return_value = system("ls -l");
    printf("コマンドの戻り値: %d\n", return_value);
    return 0;
}

出力

コマンドの戻り値: 0
