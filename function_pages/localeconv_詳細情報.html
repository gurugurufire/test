localeconv

プロトタイプ

struct lconv* localeconv(void);

詳細説明

localeconv関数は、現在のロケールの書式設定情報を含む構造体を返します。この構造体には、数値の書式、通貨の書式などの情報が含まれています。

使用例

コード

#include <stdio.h>
#include <locale.h>
#include <stdlib.h>

int main() {
    struct lconv* lc;

    setlocale(LC_ALL, "");
    lc = localeconv();
    printf("通貨記号: %s\n", lc->currency_symbol);
    printf("小数点記号: %s\n", lc->decimal_point);

    return 0;
}

出力

通貨記号: $
小数点記号: .
