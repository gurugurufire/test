exit

プロトタイプ

void exit(int status);

詳細説明

exit関数は、プログラムの実行を終了し、制御をオペレーティングシステムに戻します。status引数は、プログラムの終了状態を表し、これはオペレーティングシステムや他のプログラムによって使用されることがあります。

使用例

コード

#include <stdio.h>
#include <stdlib.h>

int main() {
    printf("プログラムを終了します。\n");
    exit(EXIT_SUCCESS); // プログラムを正常に終了
}

出力

プログラムを終了します。
