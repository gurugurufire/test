realloc

プロトタイプ

void* realloc(void* ptr, size_t new_size);

詳細説明

realloc関数は、指定されたポインタで指し示されるメモリブロックのサイズを新しいサイズに変更します。新しいサイズが元のサイズより大きい場合、追加のメモリが割り当てられます。新しいサイズが元のサイズより小さい場合、余分なメモリは解放されます。この関数は新しいメモリブロックへのポインタを返します。

使用例

コード

#include <stdio.h>
#include <stdlib.h>

int main() {
    int* ptr = malloc(10 * sizeof(int));  // 10個分のintサイズのメモリを割り当てる
    if (ptr == NULL) {
        perror("malloc");
        return 1;
    }

    // メモリのサイズを20個分のintサイズに変更する
    ptr = realloc(ptr, 20 * sizeof(int));
    if (ptr == NULL) {
        perror("realloc");
        return 1;
    }

    // 使用後のメモリを解放する
    free(ptr);
    return 0;
}
