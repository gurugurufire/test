isalpha

プロトタイプ

int isalpha(int c);

詳細説明

isalpha関数は、引数として与えられた文字がアルファベット（a-z または A-Z）であるかどうかをテストします。もしアルファベットであれば、0以外の値（真）を返し、そうでなければ0（偽）を返します。

使用例

コード

#include <stdio.h>
#include <ctype.h>

int main() {
    char c;
    printf("文字を入力してください: ");
    scanf("%c", &c);

    if(isalpha(c)) {
        printf("'%c' はアルファベットです。\n", c);
    } else {
        printf("'%c' はアルファベットではありません。\n", c);
    }

    return 0;
}
