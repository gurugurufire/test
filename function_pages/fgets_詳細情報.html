fgets

プロトタイプ

char *fgets(char *str, int n, FILE *stream);

詳細説明

fgets関数は、指定されたファイルストリームから文字列を読み込み、それをstrで指定された文字列に格納します。nは読み込む最大文字数を指定し、ファイルの終端に達するか改行文字を読み込むまで文字を読み込みます。成功した場合はstrを返し、失敗した場合はNULLを返します。

使用例

コード

#include <stdio.h>

int main() {
    char buffer[100];
    printf("文字列を入力してください: ");
    fgets(buffer, 100, stdin);
    printf("入力された文字列: %s", buffer);
    return 0;
}

出力

文字列を入力してください: こんにちは
入力された文字列: こんにちは
