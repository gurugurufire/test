log

プロトタイプ

double log(double x);

詳細説明

log関数は、引数として与えられた実数値xの自然対数（底はネイピア数e）を計算して返します。この関数はヘッダーファイルに定義されています。

使用例

コード

#include <stdio.h>
#include <math.h>

int main() {
    double number = 2.718281828459045;
    double result = log(number);
    printf("log(%lf) = %lf\n", number, result);
    return 0;
}

出力

log(2.718282) = 1.000000
