clock

プロトタイプ

clock_t clock(void);

詳細説明

clock関数は、プログラムの開始からの処理時間（CPU時間）を測定するために使用されます。この関数は<time.h>ヘッダーに定義されています。返り値はclock_t型で、時間を表す値です。

使用例

コード

#include <stdio.h>
#include <time.h>

int main() {
    clock_t start, end;
    double cpu_time_used;
    
    start = clock();
    // 何か時間のかかる処理
    end = clock();
    cpu_time_used = ((double) (end - start)) / CLOCKS_PER_SEC;
    
    printf("処理時間: %f秒\n", cpu_time_used);
    return 0;
}

出力

処理時間: x.xxxxxx秒
