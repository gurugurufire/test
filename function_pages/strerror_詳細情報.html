strerror

プロトタイプ

char *strerror(int errnum);

詳細説明

strerror関数は、エラー番号errnumに対応するエラーメッセージを返します。この関数は、エラー処理で使用されます。

使用例

コード

#include <stdio.h>
#include <string.h>
#include <errno.h>

int main() {
    FILE *fp = fopen("nonexistentfile.txt", "r");
    if (fp == NULL) {
        int errnum = errno;
        printf("エラー発生: %s\n", strerror(errnum));
    } else {
        // ファイル操作
        fclose(fp);
    }
    return 0;
}

出力

エラー発生: No such file or directory
